RAG Chatbot App - EchoResume
Python / RAG / StreamLit / ChromaDB
This project is a multimodal chatbot built using Retrieval-Augmented Generation (RAG), capable of processing resumes and answering queries in both text and audio formats. It leverages Chroma DB for vector storage, LangChain and LangSmith for managing LLM workflows, and OpenAI for generating context-aware responses. Users can interact via voice using Google Speech APIs, making the experience seamless and accessible. The app is built with Streamlit, providing an intuitive UI for uploading resumes and interacting with the bot.
